Frequently asked questions

Data is available at ZIP code and census tract levels where available. Geographic precision varies by source dataset and reporting requirements.

All data includes source attribution and methodology documentation. We maintain an audit trail that links back to original sources, making it possible to verify and reproduce our findings.

Yes. The project is designed for journalists, researchers, and policymakers. All datasets are available for download, and we provide stable links for citation. Please attribute Sunlight Ledger when using our data.

Update frequency depends on source data availability. We document versioning and update dates for each dataset. Check the methodology documentation for specific update schedules.

Sunlight Ledger prioritizes transparency and reproducibility. Definitions, assumptions, and caveats are built into the interface itself, not buried in appendices. Every number can be traced back to its source with clear methodology.
